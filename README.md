# SCANOCR

SCANOCR adalah suatu aplikasi yang dapat melakukan scanning foto menjadi sebuah dokumen, dan melakukan konversi ke berbagai format digital seperti pdf, jpeg, docx, dll


# Module

```bash
pip install PyQt5
```
